# Защищенный Электронный Документооборот Конфиденциальных Документов (ЗЭДКД)

## Описание проекта
Веб-приложение для безопасного документооборота с криптографической защитой на основе отечественных стандартов.

## Особенности
- Использование алгоритмов МАГМА, Стрибог и ГОСТ 34.10-2018
- Поддержка ролевой модели (Администратор, Пользователь, Контролер)
- Автоматическая маршрутизация документов
- Полная криптографическая защита данных
