About Key Management

In this system, each user has a pair of cryptographic keys:

  • Public Key: Used for encrypting documents that the user should be able to read
  • Private Key: Used for decrypting documents and creating digital signatures. Stored encrypted with a system master key.

Key reset will generate new key pairs for the user. This should be done if a user's private key is compromised.
